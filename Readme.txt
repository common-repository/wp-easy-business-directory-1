WP EASY BUSINESS DIRECTORY

== Description ==

WordPress Easy Business Directory is a simple two category lever and quick-setup business directory plugin for WordPress.


== Installation ==

1. Upload the wp-easy-business-directory folder to your '/wp-content/plugins/' directory.
2. Activate the plugin via the Plugins menu in WordPress.
3. Create and publish a new page with this tag in the body: [liste]
4. Take the page permalink , go to the wp easy business directory page and paste the permalink into "Business Wordpress Permalink" option field
5. Create your first category and your first subcategory.
6. Create yout first company listing
7. ÎEnjoy the plugin ;)
